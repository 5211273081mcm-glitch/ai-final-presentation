## v3 策略（当前）

**不再整画布放大**（避免边缘信息被裁切）。

| 手段 | 说明 |
|---|---|
| **满屏 fit** | 与原版相同缩放，保证 1920×1080 内容完整可见 |
| **压缩边距** | 安全区 88→56px，标题区上移 |
| **统一 token +18%** | 全部 `--fs-*` 同步提升，不单点放大 |
| **藏次要文案** | 面包屑、路径副说明、会话统计等（核心信息保留） |
| **对比/字重/线宽** | 文字更亮、标题 800、路径 6px、边框 3–4px |
